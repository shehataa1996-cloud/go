# 🏆 eDahab | إي دهب

**إي دهب** هو تطبيق ويب متطور (PWA) مصمم لتقديم متابعة دقيقة ولحظية لأسعار الذهب، الفضة، والعملات في السوق المصري والعالمي. يتميز التطبيق بتصميم عصري فخم وتجربة مستخدم سلسة تحاكي تطبيقات الهاتف الأصلية.

![eDahab Preview](https://images.unsplash.com/photo-1581091226825-a6a2a5aee158?auto=format&fit=crop&w=800&q=80)

## ✨ المميزات الرئيسية

- **🔄 تحديثات لحظية:** جلب أسعار الذهب والفضة مباشرة من البورصات العالمية مع معادلات تصحيح للسوق المحلي.
- **⚖️ حاسبة السبائك الذكية:** حساب تكلفة السبائك شاملة المصنعية وضريبة القيمة المضافة بدقة عالية.
- **💱 محول العملات:** دعم أسعار الصرف الرسمية بالإضافة إلى رصد تقديري لأسعار "السوق الموازي".
- **🗞️ مركز الأخبار:** تغذية إخبارية اقتصادية حية من مصادر موثوقة مثل سكاي نيوز عربية والعربية نت.
- **📱 دعم PWA:** إمكانية تثبيت التطبيق على الشاشة الرئيسية للهاتف والعمل في وضع "بدون اتصال" (Offline Mode).
- **🔐 نظام مستخدمين متكامل:** تسجيل الدخول عبر Google و Facebook باستخدام Firebase لتقديم تجربة مخصصة.
- **🗺️ دليل الصاغة:** منصة تربط بين أصحاب المحلات والعملاء (قيد التطوير).

## 🛠️ التقنيات المستخدمة

- **Frontend:** HTML5, CSS3 (Custom Variables, Grid, Flexbox), Vanilla JavaScript (Modular ES6).
- **Backend as a Service:** [Firebase](https://firebase.google.com/) (Authentication, Firestore, Storage).
- **APIs:**
  - [GoldAPI.io](https://www.goldapi.io/) لأسعار المعادن.
  - [Open ER-API](https://open.er-api.com/) لأسعار العملات.
  - [RSS2JSON](https://rss2json.com/) لمعالجة الأخبار.
- **Deployment:** GitHub Pages.

## 🚀 طريقة التشغيل المحلي

1.  قم بعمل Clone للمستودع:
    ```bash
    git clone https://github.com/YOUR_USERNAME/eDahab.git
    ```
2.  افتح المجلد باستخدام **VS Code**.
3.  تأكد من تحديث إعدادات Firebase في ملف `index.html`:
    ```javascript
    const firebaseConfig = {
      apiKey: "YOUR_API_KEY",
      // ... باقي الإعدادات
    };
    ```
4.  قم بتشغيل المشروع باستخدام إضافة **Live Server** لضمان عمل ميزات الـ Fetch و Service Worker.

## 📁 هيكل المشروع

- `index.html`: الملف الرئيسي وهيكل التطبيق.
- `style.css`: التنسيقات والألوان الملكية.
- `main.js`: المتحكم الرئيسي في التنقل والوظائف العامة.
- `gold.js` / `silver.js` / `currency.js`: منطق جلب البيانات والحسابات.
- `sw.js`: ملف Service Worker لدعم العمل بدون إنترنت.
- `manifest.json`: إعدادات تثبيت التطبيق على الهاتف.
